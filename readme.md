# 1. pyside6 모듈 설치
- `python --version : Python 3.11.1`
- `pip install pyside6`

- batch file (for windows)
    * 1. make_venv(win).bat
        * 가상환경 세팅 배치파일
    * 2. open_code.bat
        * vscode 실행 배치파일
    * 3. open_pydesigner.bat
        * pyside6 designer 실행 배치파일
